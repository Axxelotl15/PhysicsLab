# PhysicsLab

A personal computational physics journal documenting my exploration of physics, mathematics, numerical methods, and scientific computing.

---

## Purpose

This repository serves as a collection of projects, simulations, notes, and experiments aimed at developing a deeper understanding of physical systems and the mathematics behind them.

My goal is not simply to write code, but to understand:

- Why physical models behave the way they do
- How numerical methods approximate physical systems
- The mathematical structures underlying physics
- The connection between theory, computation, and simulation

## My Reasoning Behind Creating this Profile

This repository is intended to function as a computational notebook and learning journal where every project documents:

1. The physical problem
2. The mathematical theory
3. The computational implementation
4. Results and observations
5. Lessons learned

The focus is understanding rather than memorization.
